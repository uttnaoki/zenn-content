## つくったもの

ボードゲーム会の集客に[TwiPla](https://twipla.jp/)がよく利用されています。
TwiPlaで告知されている福岡のボードゲーム会のイベントをカレンダーで確認するWebサイトを作成しました。
[ボードゲームカレンダーのリンク](https://boardgame-calendar.vercel.app/)

![](/images/249da7e82cb25c/web-site-capture.png =200x)

## 技術スタック

- Next.js
- supabase（DB）
- vercel（ホスティング・cron）

## supabaseがとても使いやすい

今回は初めてsupabaseを使ってみました。
無料枠でも十分な容量があり、管理画面がとても使いやすいです。
Next側からDBを触るコードも簡単で、AIを使えば一瞬でバックエンドを実装できます。
```ts
  // イベント数を取得
  const { count } = await supabase
    .from("events")
    .select("*")
```

課金すればDBの環境もブランチという形で複数作ることができるようで、
検証環境も整理しやすそうです。（無課金の場合はmainブランチのみ使用可能）
![](/images/249da7e82cb25c/supabase.png)

## おわりに

supabaseとvercelを使って無料で簡単にサイトを公開できました。
イベント情報がカレンダーに整理され、今後も使っていける便利なものができました。

ページの読み込みが遅いため、パフォーマンスについて勉強しながら改善していきたいです。
また、Googleから当サイトが検索できないため、SEOの勉強もやっていきたいです。
